Healenium API Specification

Self-healing testing tool that helps to maintain web tests by automatically healing broken locators

Base URL: /healenium

Selector Management

POST /healenium Save Selector

Save information about a successfully found element

Request Body
SelectorRequestDto 
{
  "locator": "//div[@id='example']",
  "className": "com.example.TestClass",
  "methodName": "testMethod",
  "command": "findElement",
  "url": "https://example.com",
  "type": "xpath",
  "enableHealing": true
}
Response
204 No Content
GET /healenium Get Reference Elements

Get last valid path for provided request

Request Parameters
  • locator: The locator string
  • className: The class name
  • methodName: The method name
  • command: The command (findElement or findElements)
  • url: The URL
Response
ReferenceElementsDto 
{
  "elements": [
    {
      "nodePathes": [...],
      "elementIds": ["id1", "id2"]
    }
  ]
}
GET /healenium/elements Get All Elements and Config

Get all saved selectors and configuration

Response
ConfigSelectorDto 
{
  "config": {...},
  "selectors": [...]
}
GET /healenium/selectors Get Selectors Page

Get all selectors for the selector.html view

Response
ModelAndView with selector data
GET /healenium/selector/all Get All Selectors

Get all selectors as a list

Response
List of SelectorRequestDto 
[
  {
    "id": "selector-id",
    "locator": "//div[@id='example']",
    "className": "com.example.TestClass",
    "methodName": "testMethod",
    "command": "findElement",
    "url": "https://example.com",
    "type": "xpath",
    "enableHealing": true
  }
]
POST /healenium/selector/status Set Selector Status

Set status (enable/disable) for healing

Request Body
SelectorDto 
{
  "id": "selector-id",
  "enable": true
}
Response
204 No Content

Healing Management

POST /healenium/healing Save Healing

Save healing result for specific selector

Headers
  • sessionKey: Session key for report association
Request Body
List of HealingRequestDto 
[
  {
    "locator": "//div[@id='example']",
    "className": "com.example.TestClass",
    "methodName": "testMethod",
    "command": "findElement",
    "url": "https://example.com",
    "type": "xpath",
    "results": [
      {
        "locator": "//div[@id='new-example']",
        "score": 0.95
      }
    ],
    "usedResult": {
      "locator": "//div[@id='new-example']",
      "score": 0.95
    }
  }
]
Response
204 No Content
GET /healenium/healing Get Healings

Get healing with their results for provided request

Request Parameters
  • locator: The locator string
  • className: The class name
  • methodName: The method name
  • command: The command
  • url: The URL
Response
Set of HealingDto 
[
  {
    "id": "healing-id",
    "locator": "//div[@id='example']",
    "results": [
      {
        "id": "result-id",
        "locator": "//div[@id='new-example']",
        "score": 0.95,
        "successHealing": true
      }
    ]
  }
]

Report Management

GET /healenium/report/{uid} Get Report by ID

Get report by ID

Path Parameters
  • uid: Report unique ID
Response
ModelAndView with report data
GET /healenium/report/data/{uid} Get Report Data

Get report data by ID

Path Parameters
  • uid: Report unique ID
Response
RecordDto 
{
  "id": "report-id",
  "name": "Report Name",
  "time": "2025-09-08T12:00:00",
  "data": [
    {
      "declaringClass": "com.example.TestClass",
      "screenShotPath": "/path/to/screenshot.png",
      "failedLocatorValue": "//div[@id='example']",
      "failedLocatorType": "xpath",
      "healedLocatorValue": "//div[@id='new-example']",
      "healedLocatorType": "xpath",
      "score": "0.95",
      "successHealing": true,
      "healingResultId": 123
    }
  ]
}
